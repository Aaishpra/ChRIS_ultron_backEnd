Feed collection¶

Read-only

This resource type refers to a collection of user-specific feed items.

A representation is served at the root of the API: /api/v1/.

In other resource representations this resource type is linked by any link relation with attribute:

“rel”: “feeds”

GET /api/v1/¶

Gets the list of feeds for the authenticated user.

Example request:

Host: localhost:8000
Accept: application/vnd.collection+json
